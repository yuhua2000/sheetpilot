# Excel MCP Server 需求文档

## 1. 项目概述

构建基于 Go 的 Excel MCP Server，让 AI 像熟练用户一样操作 Excel。

用户仅需自然语言即可完成复杂 Excel 工作，无需理解 Sheet、单元格坐标、公式等细节。

```text
打开 report.xlsx
帮我整理这个表
新增一列利润率
按地区统计销售额
生成图表
```

---

## 2. 核心设计原则

1. **AI First** - Tool 负责提取数据返回给 AI，AI 来理解和决策
2. **批量优先** - 减少 MCP 往返，一次批量写入而非万次 `set_cell`
3. **易扩展** - 模块化架构，后续可支持 CSV、Google Sheets 等

---

## 3. 技术选型

| 组件 | 选型 | 说明 |
|------|------|------|
| 语言 | Go 1.24+ | 单二进制、跨平台、高性能 |
| Excel 引擎 | [excelize](https://github.com/qax-os/excelize) | Excel 读写 |
| MCP SDK | [mcp-go](https://github.com/mark3labs/mcp-go) | MCP 协议实现 |
| CLI 框架 | cobra | 子命令管理 |

### 文件格式

- 当前：`.xlsx`
- 后续：`.xls`（自动识别）
- 暂不支持：`.csv`、`.ods`、Google Sheets

### 通信模式

- stdio（默认）
- SSE

### CLI 结构

```bash
sheetpilot version
sheetpilot serve                    # stdio 模式
sheetpilot serve --transport sse    # SSE 模式
```

### 跨平台

单二进制发布，支持 Linux/macOS/Windows（amd64/arm64）。

## 5. 需求阶段划分

### 阶段 1：基础设施 + 基础操作

**目标**：搭建骨架，实现核心 Excel 操作

| 模块 | 功能 | 状态 |
|------|------|------|
| 架构 | CLI + MCP Server + stdio + SSE | ✅ |
| Workbook | `open_workbook`、`save_workbook`、`close_workbook`、`list_workbooks`、`get_workbook_info` | ✅ |
| Sheet | `list_sheets`、`create_sheet`、`rename_sheet`、`delete_sheet`、`copy_sheet`、`set_active_sheet`、`get_sheet_info` | ✅ |
| 单元格 | `get_cell`、`set_cell`、`clear_cell` | ✅ |
| Range | `get_range`、`set_range`、`append_rows`、`insert_rows`、`delete_rows`、`insert_columns`、`delete_columns` | ✅ |

**交付标准**：可启动 MCP Server，打开/编辑/保存 Excel 文件

---

| | `column_types` | 每列采样数据 + 推断类型 | ✅ |
| | `sheet_overview` | 行数、列数、列名、列类型、样本数据 | ✅ |
| 数据操作 | `add_computed_column` | 新增计算列 | ✅ |
| | `fill_formula_column` | 公式批量填充 | ✅ |
| | `fill_missing_values` | 补全空值 | ✅ |
| | `replace_values` | 批量替换 | ✅ |
| | `cleanup_sheet` | 自动清洗 | ✅ |
| | `deduplicate` | 去重 | ✅ |
| | `sort_table` | 排序 | ✅ |
| | `filter_rows` | 条件筛选 | ✅ |
| | `group_by` | 聚合统计（使用 Excel 公式） | ✅ |
| | `split_sheet` | 按条件拆分 | ✅ |
| | `merge_sheets` | 合并 sheet | ✅ |
| 公式 | `get_formula`、`set_formula` | 公式读写 | ✅ |
| | `fill_formula_column` | 公式批量填充 | ✅ |
| Range 操作 | `copy_range` | 复制区域到其他位置 | ✅ |
| | `move_range` | 移动区域 | ✅ |

| 基础样式 | `auto_fit_columns`、`auto_fit_rows`、`set_number_format` | 格式调整 | ✅ |

**交付标准**：AI 能获取表格结构信息，支持数据处理和常见 Excel 操作

## 6. 性能要求

| 普通操作 | < 500ms |
| 批量操作 | < 5s |

## 7. 错误处理

统一错误结构：

```json
{
  "error": {
    "code": "sheet_not_found",
    "message": "sheet 不存在"
  }
}
```

错误类型：`workbook_not_found`、`sheet_not_found`、`invalid_formula`、`invalid_range`、`permission_denied`、`file_locked`

---

## 8. 最终目标

> 让 AI 真正"会用 Excel"，而不是"会调 Excel API"。

用户像在和精通 Excel 的助手聊天，自然语言描述需求，AI 自动完成所有操作。
